# Android App Development Notes (Java)

## Overview
This repository contains structured notes on Android app development using Java. It serves as a quick reference for understanding core concepts, components, and best practices required to build Android applications.

The content is organized to support learning, revision, and practical implementation.

## Contents
- Introduction to Android  
- Android Architecture  
- Activities and Activity Lifecycle  
- Intents and Intent Filters  
- UI Components and Layouts  
- RecyclerView and Adapters  
- Fragments  
- Storage Options (SharedPreferences, SQLite)  
- Networking Basics  
- Permissions  
- Services and Broadcast Receivers  
- Debugging and Testing  

## Purpose
The goal of this repository is to:
- Provide a centralized collection of Android development notes  
- Enable quick revision for exams and interviews  
- Act as a reference while building Android applications  

## Tech Stack
- Java  
- Android SDK  
- Android Studio
